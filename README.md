# Simple Web

A simple static website built using HTML and CSS.
This project demonstrates a basic web page structure with organized assets such as stylesheets and images.

---

## 📌 Description

This project is a simple static website created to practice fundamental web development skills.

The website includes:

* A main HTML page
* Custom CSS styling
* Image assets for content display
* Organized folder structure using an `assets` directory

This project is suitable as a beginner portfolio example for front-end development.

---

## 🚀 Features

* Static web page
* Custom CSS styling
* Image-based content
* Organized project structure
* Lightweight and easy to run

---

## 🛠 Tech Stack

* HTML
* CSS

---
